# Stellar Nebula

A protostellar implementation.

## Description

Provides translation capabilities for Protostellar to legacy protocols.

## Getting Started

### Dependencies

- Go @ v1.18+
- protoc @ v3+
- protoc-gen-go @ v1.28+
- protoc-gen-go-grpc @ v1.2+

### Executing program

```
> go generate
> go build
> ./stellar-nebula
```

or alternatively while developing:

```
> go generate
> go run ./
```
